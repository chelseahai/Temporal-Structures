NYC Heat Vulnerability Visualizations

1. Line Chart – HVI Over Time by ZIP

This line chart displays the Heat Vulnerability Index (HVI) from 2016 to 2020 for three ZIP codes. It visualizes how vulnerability levels changed across time in each area.

2. Calendar Heatmap – Daily HVI

This calendar heatmap shows synthetic daily HVI values across the year 2020. Each square represents one day, with color intensity indicating vulnerability level.

3. Radial Chart – Monthly HVI by ZIP

This radial chart encodes monthly HVI data for each ZIP code in a circular layout, highlighting seasonal patterns and differences across locations.

4. Choropleth Map – 2020 HVI by ZIP

This choropleth map shows spatial variation in HVI for NYC ZIP codes in 2020. Darker colors represent higher vulnerability.
